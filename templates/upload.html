pyflame-server demo

start python to profile

./pyflame -t python3.5 -c "print(sum(i for i in range(1000000))) > profile.txt

or attach python pid to profile, -s seconds -p pid

sudo ./pyflame -s 5 -p 28947 > profile.txt

upload to pyflame-server

curl -F "file=@profile.txt" http://172.22.22.230:5000/upload
